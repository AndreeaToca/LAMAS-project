This website presents the information about the project for the course Multi-Agent Systems course of the Rijksuniversiteit Groningen in the year 2020/2021. We looked at the "23-logicians" riddle created by Edwin Meyer. We propose the solution using epistemic logic, as well as an application that solves the riddle.
